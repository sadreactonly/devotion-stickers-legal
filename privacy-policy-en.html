Privacy Policy - Devotion Stickers

Last updated: [DATE]

1. Data We Collect

When using the Devotion Stickers app, we collect the following data:

  • Mandatory data: Email address, username, password.
  • Optional data: Phone number (if the user chooses to use the SMS contact option).
  • Account data: Sticker status (collected, duplicates), event information for trading.

2. Authentication Method

Users can create an account using an email address and password, as well as through Google or Apple login.

3. Data Storage and Protection

All data is stored in the Firebase Firestore database, while user authentication is handled through Firebase Authentication. Devotion Stickers employs appropriate security measures to protect user data.

4. Ads and Monetization

The app displays ads that may use cookies and other tracking technologies from third parties.

5. Sticker Trading and SMS Functionality

Users can:

  • Create events for sticker trading.
  • Contact other users via SMS.
  • Generate a QR code for easier live sticker trading.

The phone number is not visible to other users but is automatically inserted into an SMS message if the user utilizes the contact option. Entering a phone number is optional but required for using the SMS function.

6. Account and Data Deletion

Users can delete their account at any time from within the app, which will immediately remove all their data from the system.

7. Tracking and Analytics

We currently do not use data tracking tools (e.g., Google Analytics), but we reserve the right to add them in the future with an update to this privacy policy.

8. User Age Requirement

Devotion Stickers is intended for users aged 12 and older.

9. User Rights and Responsibilities

Users are responsible for the content they share and the messages they send through the app. Devotion Stickers does not monitor communication between users and assumes no responsibility for their agreements.

10. Reporting Violations

Users currently cannot report inappropriate behavior, but we plan to introduce this feature in the future.

11. Changes to the Privacy Policy

We reserve the right to update this privacy policy. Users will be notified of changes via the app.
